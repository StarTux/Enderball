# Enderball

Enderball plugin.

# Areas

Areas file name: `Enderball`

Required are as per `GameBoard.java`:
- `area` (unique, used for scanning)
- `field` (unique, where players and the ball can go)
- `kickoff` (one block, unique)
- `goal` (two, labeled `red` and `blue`)
- `spawn` (two, labeled `red` and `blue`)
- `viewers` (multiple, at least one, where viewers will be teleported to)
